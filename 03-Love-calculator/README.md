# Love Calculator

A simple JavaScript project that calculates a fun love percentage based on the length of two names entered by the user.

## Features

* Enter two names
* Calculates a random love percentage
* Displays the result without refreshing the page

## Technologies Used

* HTML
* CSS
* JavaScript

## What I Learned

* Form Events (`submit`)
* Event Listeners
* `preventDefault()`
* DOM Manipulation
* Getting Input Values
* Updating Text Dynamically
