# To-Do List App 📝

A simple and interactive To-Do List application built with HTML, CSS, and JavaScript. Users can add tasks, mark them as completed, and delete them dynamically without refreshing the page.

## Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Prevent empty task submission
* Dynamic DOM manipulation

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Project Structure

```text
project/
│
├── index.html
├── style.css
└── script.js
```

## Concepts Practiced

* DOM Manipulation
* Event Listeners
* Form Handling
* Dynamic Element Creation
* Event-Driven Programming
