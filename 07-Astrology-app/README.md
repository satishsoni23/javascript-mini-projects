# Zodiac Personality Generator

A fun and interactive web application built with **HTML, CSS, and JavaScript** that generates a personalized zodiac profile based on the user's name and date of birth.

## ✨ Features

* Displays a zodiac sign.
* Generates personalized compliments and predictions.
* Provides motivational recommendations.
* Dynamic content generation using JavaScript arrays and indexing.

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
