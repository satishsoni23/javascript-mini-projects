# ⏳ Olympics 2028 Countdown Timer

A simple **Countdown Timer** built with **HTML, CSS, and JavaScript** that calculates the remaining time until the **2028 Summer Olympics** and displays it in days, hours, minutes, and seconds.

## ✨ Features

* Calculates the time remaining until July 14, 2028.
* Displays countdown in:

  * Days
  * Hours
  * Minutes
  * Seconds
* Uses JavaScript `Date` and timestamp calculations.
* Beginner-friendly project to understand time manipulation and arithmetic operations.

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🚀 How It Works

1. Gets the current timestamp using `Date.now()`.
2. Creates the Olympics date using `new Date()`.
3. Calculates the difference between the two timestamps.
4. Converts the remaining milliseconds into days, hours, minutes, and seconds.
5. Displays the countdown dynamically on the webpage.
