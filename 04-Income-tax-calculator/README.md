# 💰 Income Tax Calculator

A simple and interactive **Income Tax Calculator** built using **HTML, CSS, and JavaScript**. This project calculates the total income tax based on different income tax slabs and displays the result instantly without reloading the page.

---

## 🚀 Features

* Calculate income tax based on predefined tax slabs.
* Instant result using JavaScript DOM manipulation.
* Prevents page refresh using `preventDefault()`.
* User-friendly interface.
* Automatically clears the input field after calculation.
* Beginner-friendly project to understand JavaScript events and conditions.

---

## 🛠️ Technologies Used

* **HTML5** – Structure of the application.
* **CSS3** – Styling and responsive design.
* **JavaScript (ES6)** – Business logic and DOM manipulation.

## 📋 How It Works

1. Enter your annual income.
2. Click the **Calculate Tax** button.
3. The application:

   * Reads the input value.
   * Determines the applicable tax slab.
   * Calculates the total tax.
   * Displays the result on the screen.
   * Resets the input field for the next calculation.

---

## 🧮 Tax Slabs Used

| Annual Income           | Tax Rate |
| ----------------------- | -------- |
| Up to ₹12,00,000        | 0%       |
| ₹12,00,001 – ₹16,00,000 | 15%      |
| ₹16,00,001 – ₹20,00,000 | 20%      |
| ₹20,00,001 – ₹24,00,000 | 25%      |
| Above ₹24,00,000        | 30%      |
