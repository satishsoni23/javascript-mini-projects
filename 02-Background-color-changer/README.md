# Background Color Changer

A simple JavaScript project that changes the webpage background color when a color button is clicked.

## Features

* Change background color with one click
* Built using Event Delegation
* Simple and beginner-friendly project

## Technologies Used

* HTML
* CSS
* JavaScript

## What I Learned

* DOM Manipulation
* Event Listeners
* Event Delegation
* `e.target`
* Changing CSS styles using JavaScript
