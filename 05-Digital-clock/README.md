# 🕒 Digital Clock

A simple and responsive **Digital Clock** built using **HTML, CSS, and JavaScript**. The application displays the current time and updates automatically every second using JavaScript's `Date` object and `setInterval()`.

## ✨ Features

* Displays the current local time.
* Updates every second in real time.
* Uses `setInterval()` for continuous updates.
* Beginner-friendly project for learning DOM manipulation and JavaScript timers.

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🚀 How It Works

1. Selects the clock container using `getElementById()`.
2. Creates a new `Date` object every second.
3. Converts the time into a readable format using `toLocaleTimeString()`.
4. Updates the UI dynamically without refreshing the page.
