# Click Animation Effect 🎉

A simple JavaScript project where clicking anywhere on the page creates a colorful circle at the mouse position. The circle contains the text **"KRSN"** and automatically disappears after 5 seconds.

## 🚀 Features

* Creates a new circle on every click.
* Displays the text **KRSN** inside the circle.
* Generates a random background color.
* Places the circle exactly where the user clicks.
* Removes the circle automatically after 5 seconds.

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (DOM Manipulation)

## 📂 Project Structure

```
project-folder/
```

## 📸 How It Works

1. User clicks anywhere on the webpage.
2. JavaScript creates a new `<div>` element.
3. A random color is assigned to the circle.
4. The circle appears at the click position.
5. After 5 seconds, the circle disappears automatically.

3. Click anywhere on the page and enjoy the animation!
