# Cricket Quiz App 🏏

A simple quiz application built using HTML, CSS, and JavaScript. Users can answer multiple-choice cricket questions and instantly receive their score after submitting the quiz.

## Features

* Multiple-choice quiz questions
* Automatic score calculation
* Displays final score instantly
* Uses JavaScript FormData API
* Resets the quiz after submission

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Concepts Practiced

* DOM Manipulation
* Event Handling
* FormData API
* Objects and Key-Value Pairs
* Loops and Conditional Statements
